### What I learned

  In this project I learned how to use the CSS grid to display elements on a webpage: 

```html
<main>
      <section class="review-1">
      <aside class="review-2">
</main>        
```

```css
main {
    width: 90vw;
    height: 95vh;
    display: grid;
    grid-template: auto / auto auto;
    justify-content: center;
    align-items: center;
}
```
